# STUDENT-PROGRESS-TRACKER-SYSTEM

**PROJECT DESCRIPTION:**

The Student Progress Tracker System is a Python-Based mini project designed  to track and evaluate a student's academic performance.
It calculates the total marks, average marks, percentage and determines pass/fail status based on user input.
This project is suitable for beginners to understand Python fundamentals such as variables,Control Statements and Functions.

**LOGIC OVERVIEW:**

1.Take a Student details(name,pin number and class) as input

2.Accept marks for multiple subjects as user input

3.Calculate total marks, average marks and percentage

4.Check pass/fail condition using conditional  and jumping statements

5.Finally the Student Progress Card is displayed

6.After the menu is showed, if you need to any changes you can enter the choice otherwise you choose the exit choice

7.If you enter the exit choice, The tracker system is exited.

**How to run?**

1.Install Python(version 3.x.x)

2.Download or clone this repository.

3.Open this file in VS code 

4.Run the program.
